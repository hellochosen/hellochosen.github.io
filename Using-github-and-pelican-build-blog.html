Using github and pelican build blog

Wed 25 September 2013

github pages

1.create a new respository in your github website . Respository name like this : username.github.io

See more , please refer to google.

pelican

1.Introduction for pelican

Pelican is a static site generator that supports Markdown and reST syntax . You can add posts and create your pages . Also it support Disqus comment system , RSS , Atom , plugins , themes , code syntax highlight , etc ...

2.Install pelican and markdown

#In ubuntu 12.04
sudo apt-get install python-pip
pip install pelican
pip install Markdown

3.create blog

mkdir blog
cd blog
#quickly install blog
pelican-quickstart

use the default configuration , press the "Enter".

4.write post

a. change directory to blog/content , write a post using Markdown syntax:

Date: 2013-09-25 
Title: My first blog
Category: Tool 
Slug: my-first-blog #url

b. change directory to blog/:

pelican -s pelicanconf.py content/

c. In dir blog/output ,you will see the outputs.

5.create a local git respository

cd output
git init
git add .
git commit -m "first commit"
git remote add origin https:://github.com/username/username.github.io.git
git pull origin master
git push origin master
